
# stock-market-analyzer

A Python-based stock market analysis tool for real-time data fetching and visualization. This project demonstrates how to use Python for financial data analysis, including fetching historical data and plotting trends.

## Features
- Fetch historical stock data using `yfinance`.
- Visualize stock price trends using `matplotlib`.
- Analyze basic market movements.

## Technologies
- Python
- Pandas
- Matplotlib
- yfinance

## Setup
1. Clone the repository:
   `git clone https://github.com/yihang880/stock-market-analyzer.git`
2. Install dependencies:
   `pip install pandas matplotlib yfinance`

## Usage
```bash
python main.py
```
